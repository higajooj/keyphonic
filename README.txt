Passos para rodar:

1. Instalar dependencias:
  1.1. Rodar `npm install` na pasta `backend/`
  1.2. Rodar `npm install` na pasta `frontend/`

2. Criar arquivos `.env` nas pastas `backend/` e `frontend/` conforme os exemplos (`.env.example`)

3. (opcional) Iniciar Postgres se nao tiver instalado
  3.1. docker compose up

4. Iniciar servidores
  4.1. Rodar `npm run start:dev` na pasta `backend/`
  4.1. Rodar `npm run dev` na pasta `frontend/`
